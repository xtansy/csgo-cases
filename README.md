# CS:GO Cases Simulator

Симулятор открытия кейсов в стиле CS:GO с мини-играми.

## 🚀 Быстрый старт

```bash
npm run dev
```

## 📦 Функционал

- **Кейсы** — 5 кейсов с разной ценой и предметами
- **Инвентарь** — хранение и продажа предметов
- **Crash** — игра с растущим множителем
- **Roulette** — ставки на цвета

## 🛠️ Стек

- React 19 + TypeScript
- Vite
- Tailwind CSS
- framer-motion
- Zustand
- React Router

## 📊 Особенности

- Стартовый баланс: $10,000
- Звуковые эффекты
- Сохранение в localStorage
- Анимации и переходы
